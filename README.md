C-ALGORITHMS
============

A curated collection of foundational C programs covering sorting, searching, mathematical operations, file handling, and user-defined data types. This repository is designed for learners, educators, and developers who want to understand core algorithmic logic in C.

Folder Structure
----------------
C-ALGORITHMS/
├── bubblesort.c
├── insertionsort.c
├── selectionsort.c
├── linearsearch.c
├── binarysearch.c
├── factorial.c
├── matrixaddition.c
├── multiplication.c
├── cmdlineargs.c
├── userdefinedunion.c
├── filecopy.c
├── string.c
├── function.c
├── complexno.c
├── mergefile.c
├── improve-linearsearch.c
├── a.out / *.exe (compiled outputs)

How to Run
----------
Compile any `.c` file using GCC:
    gcc filename.c -o outputname
    ./outputname

Example:
    gcc bubblesort.c -o bubblesort
    ./bubblesort
